Production rules

  • 1920 x 1080 screen size highly recommended
  • Production must be run in real time and not just be a pre-rendered video
    • Your production may, however, use pre-rendered material
  • You must own or have the permission to use all the material in your production
  • Organizers reserve the right to augment or modify the competition rules if deemed necessary
  • The size of the production must be under 300mb
    • We recommend that the production is much under this size
    • If the size of your production is too large, talk with the organizers to figure things out
  • The production must be uploaded to http://graffathonpartysystem.dy.fi before the deadline
  • Teams should have 1-3 members.
  • Sources will be released

Machine specs

  • Xeon E3-1230 v3 (about Core i5-4670K)
  • GTX1060 (6GB)
  • 8 GB RAM
